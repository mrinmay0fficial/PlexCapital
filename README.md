# PlexCapital 🪙 
**Navigating the New Economy through Data and Precision.**

PlexCapital is a high-performance, data-driven financial authority hub designed for the 2026 economic landscape. We bridge the gap between complex global tax laws and actionable wealth-management strategies for digital nomads, remote professionals, and global entrepreneurs.

## 🚀 Vision
In an era of shifting jurisdictions and digital currencies, "standard" financial advice is obsolete. PlexCapital leverages modern web technologies and financial modeling to provide users with real-time clarity on their purchasing power and tax liabilities.

## 🛠️ Tech Stack
* **Architecture:** Static Site Generation (SSG) for maximum performance and security.
* **UI/UX:** Custom-built modern CSS framework focused on "Trust-First" design (Midnight/Teal palette).
* **Engine:** JavaScript-based financial models for precise Net Income Optimization and PPP (Purchasing Power Parity) simulations.
* **Hosting:** Decentralized delivery via GitHub Pages.

## 📊 Core Features
- **Net Income Optimizer:** A high-precision calculator that models effective tax rates across top-tier nomad jurisdictions (UAE, USA, UK, Portugal).
- **Tax Atlas (Coming Soon):** A database of global tax treaties and digital nomad visa requirements.
- **Wealth-Management Guides:** Peer-reviewed content focusing on YMYL (Your Money Your Life) authority standards.

## 🛠 Installation & Local Development
Since this is a static project, no complex installation is required.
1. Clone the repo: `git clone https://github.com/[YOUR-USERNAME]/PlexCapital.git`
2. Open `index.html` in any modern browser.
